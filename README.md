![Image of Screen](https://github.com/overedge/FDF/blob/master/screen.png?raw=true)

# Tested for mac user (El capitan realease)

### Installation
```sh
$ git clone https://github.com/overedge/FDF FDF
$ cd FDF
$ make
$ make clean
```
### Usage
```sh
$ ./ft_select [file1] [file2] [...]
```
### Commands
| Usage       | Commands |
| ------------- | ------------- |
| SELECT  | SPACE  |
| CHOOSE  | ENTER  |
| MOVE  | ARROW ← ↑ →  |
